# Prospect – Athlete Media Promotion Landing Page

---

## Project Description

Prospect is a fictional sports media agency created to help high school and lower-division college athletes build a stronger personal brand and gain attention from Division I programs. The landing page promotes media packages designed to showcase an athlete’s skill, story, and personality through photography, video, highlight reels, and branded content.

This website was designed as a single-page landing page that introduces the brand, displays athlete work, explains the agency’s process, and presents three media packages for potential clients.

---

## Features Implemented

This project includes the following features:

- Semantic HTML5 structure using `header`, `nav`, `main`, `section`, and `footer`
- Navigation links that scroll to different sections of the page
- Hero section with background image overlay
- Responsive image gallery using CSS Grid
- “How It Works” section to explain the service process
- Call-to-action section introducing the package options
- Interactive package cards with a flip animation on hover
- Footer with internal navigation links
- CSS variables for colors, spacing, typography, and transitions
- Responsive design using media queries for tablet, mobile, and small mobile screens
